# Titulo: Calculadora e Tabuada

## Descrição:

Este é um projeto que consiste em uma **calculadora simples** com funcionalidades básicas (soma, subtração, 
multiplicação, divisão, etc.), além de um **modo científico** que oferece operações mais avançadas 
(funções trigonométricas, logaritmos, exponenciação). O projeto também inclui uma **ferramenta de tabuada** 
que permite ao usuário inserir um número e visualizar sua tabuada de 1 a 10.

- **Modo Padrão**: Para cálculos básicos.
- **Modo Científico**: Para cálculos avançados, como funções trigonométricas e logaritmos.
- **Tabuada**: Gera a tabuada de um número inserido pelo usuário.

## Uso:

1. Abra o arquivo `index.html` em seu navegador para usar a calculadora.
2. A calculadora possui dois modos de operação:
   - **Modo Padrão**: Realiza operações matemáticas básicas.
   - **Modo Científico**: Ative clicando no botão "Mudar para Científica", permitindo operações avançadas como seno, 
   cosseno, tangente, logaritmos e exponenciação.
3. Para acessar a **Tabuada**, clique no botão "Tabuada" para abrir a página onde você pode inserir um número 
e visualizar sua tabuada de 1 a 10.
4. O histórico de cálculos realizados na calculadora pode ser acessado clicando no botão **Histórico**.
